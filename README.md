# Nginx, PHP, MySQL
###1- سو كلون للريبو هذي

`git clone https://github.com/M75/my-php-stack.git`

###2- بعده انتقل للمجلد
`cd my-php-stack`

###3- الخطوة الاخيرة من داخل المجلد شغل الستاك

`docker-compose up -d`
